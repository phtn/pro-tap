To get the counterpart chord of a note, you typically want to build a **triad** (3-note chord) or other chord types. Here's how to construct them:

## Basic Chord Types

**Major Chord** = Root + 4 semitones + 3 semitones
- C Major: `['C4', 'E4', 'G4']`
- G Major: `['G4', 'B4', 'D5']`

**Minor Chord** = Root + 3 semitones + 4 semitones
- C Minor: `['C4', 'Eb4', 'G4']` (Eb = E flat)
- A Minor: `['A4', 'C5', 'E5']`

**Diminished Chord** = Root + 3 semitones + 3 semitones
- C Diminished: `['C4', 'Eb4', 'Gb4']`

**Augmented Chord** = Root + 4 semitones + 4 semitones
- C Augmented: `['C4', 'E4', 'G#4']`

## Practical Implementation

Here's a TypeScript function to generate chords:

```typescript
type ChordType = 'major' | 'minor' | 'diminished' | 'augmented' | 'major7' | 'minor7';

const NOTES = ['C', 'C#', 'D', 'D#', 'E', 'F', 'F#', 'G', 'G#', 'A', 'A#', 'B'];

function getChord(rootNote: string, chordType: ChordType): string[] {
  // Parse note (e.g., "C4" -> note: "C", octave: 4)
  const octave = parseInt(rootNote.slice(-1));
  const note = rootNote.slice(0, -1);
  const rootIndex = NOTES.indexOf(note);

  // Interval patterns (in semitones)
  const intervals: Record<ChordType, number[]> = {
    major: [0, 4, 7],
    minor: [0, 3, 7],
    diminished: [0, 3, 6],
    augmented: [0, 4, 8],
    major7: [0, 4, 7, 11],
    minor7: [0, 3, 7, 10]
  };

  return intervals[chordType].map((interval: number) => {
    const noteIndex = (rootIndex + interval) % 12;
    const octaveOffset = Math.floor((rootIndex + interval) / 12);
    return `${NOTES[noteIndex]}${octave + octaveOffset}`;
  });
}

// Usage:
getChord('C4', 'major');    // ['C4', 'E4', 'G4']
getChord('C4', 'minor');    // ['C4', 'D#4', 'G4']
getChord('A4', 'minor7');   // ['A4', 'C5', 'E5', 'G5']
```

## Playing Chords with Tone.js

To play these chords in your context:

```typescript
const playChord = useCallback((notes: string[], duration: string): void => {
  if (!synth || !isStarted) return;
  synth.triggerAttackRelease(notes, duration);
}, [synth, isStarted]);

// Usage:
playChord(['C4', 'E4', 'G4'], '2n'); // Plays C major chord
```

The most common "counterpart" is the **relative minor/major**:
- C Major's relative minor = A Minor
- A Minor's relative major = C Major

They share the same notes but have different emotional qualities (major = happy, minor = sad).
